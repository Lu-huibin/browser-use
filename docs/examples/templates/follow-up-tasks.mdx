## 链条特工任务

保持浏览器会话活跃，并将多个任务串联起来。非常适合对话式工作流程或多步骤流程。

```python
from dotenv import load_dotenv

from browser_use import Agent, Browser


load_dotenv()

import asyncio


async def main():
	browser = Browser(keep_alive=True)

	await browser.start()

	agent = Agent(task='search for browser-use.', browser_session=browser)
	await agent.run(max_steps=2)
	agent.add_new_task('return the title of first result')
	await agent.run()

	await browser.kill()

asyncio.run(main())
```

## 工作原理

1. **持久浏览器**：“BrowserProfile（keep_alive=True）”阻止浏览器在任务间关闭
2. **任务链**：使用“agent.add_new_task（）”添加后续任务
3. **上下文保持**：代理在任务间维护内存和浏览器状态
4. **互动流程**：非常适合对话式界面
5. **拆解长流量**：如果流量非常长，可以保持浏览器存活并向其发送新代理。

<Note>
浏览器会话在整个链中保持活跃，保留所有 Cookie、本地存储和页面状态。</Note>
