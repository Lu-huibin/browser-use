## 浏览器（BrowserSession）

主浏览器会话管理器。

### 关键方法

```python
from browser_use import Browser

browser = Browser()
await browser.start()

# Page management
page = await browser.new_page("https://example.com")
pages = await browser.get_pages()
current = await browser.get_current_page()
await browser.close_page(page)

# To stop the browser session
await browser.stop()
```

## 页面

浏览器标签/iframe用于页面级作。

### 导航
- 'goto（url： str）' - 导航至URL
- 'go_back（）'， 'go_forward（）'， 'reload（）' - 历史导航

### 元素寻找
- 'get_elements_by_css_selector（选择器：str） -> list[Element]' - CSS 选择器
- 'get_element（backend_node_id： int） -> Element' - 按 CDP 节点 ID 表示
- 'get_element_by_prompt（提示词：力量，LLM）-> 元素 |“无”——AI驱动
- 'must_get_element_by_prompt（提示词：力量，LLM）->元素' - AI（未找到则提升）

### JavaScript 与控制
- 'evaluate（page_function： str， *args） -> str' - 执行JS（箭头函数格式）
- 'press（key： str）' - 发送键盘输入（“Enter”、“Control+A”）
- 'set_viewport_size（宽度：int，高度：int）' - 设置视口
- '截图（format='jpeg'， quality=None） -str' > - 截取

### 信息
- 'get_url（） -> str'， 'get_title（） -> str' - 页面信息
- “鼠标 -> 鼠标” - 获取鼠标界面

### 人工智能功能
- 'extract_content（提示词：str，structured_output：type[T]， LLM）-> T' - 提取数据

## 元素

单个DOM元素的交互。

### 互动
- 'click（button='left'， click_count=1， modifiers=None）' - 点击元素
- 'fill（text： str， clear=True）' - 填充输入
- 'hover（）'， 'focus（）' - 鼠标/焦点作
- 'check（）' - 切换复选框/无线电
- 'select_option（值：str | list[str]）' - 选择下拉选项
- 'drag_to（目标：元素 |Position）' - 拖放

### 地产
- 'get_attribute（名称：STR） -> STR |无 - 获得属性
- 'get_bounding_box（） -> BoundingBox |无 - 位置/体型
- 'get_basic_info（） -> ElementInfo' - 完整的元素信息
- 'screenshot（format='jpeg'） -> str' - 元素截图

## 老鼠

基于坐标的鼠标作。

### 行动
- 'click（x： int， y： int， button='left'， click_count=1）' - 点击坐标
- 'move（x： int， y： int， steps=1）' - 移动鼠标
- 'down（button='left'）'， 'up（button='left'）' - 按下/释放按钮
- 'scroll（x=0， y=0， delta_x=None， delta_y=None）' - 坐标处滚动
